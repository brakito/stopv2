STOP V2

¿Cómo jugar?

El sistema mostrará 4 categorías aleatorias. Para cada una, se indicará si debes hacer un dibujo o escribir una palabra.

Cada categoría tendrá una letra asignada. Tu dibujo o palabra debe comenzar con esa letra.

El juego termina cuando se acaba el tiempo o cuando alguien dice "Stop" (significa que completó todos los cuadros y los demás no pueden seguir dibujando o escribiendo).

Puntaje:

  • 100 puntos: Si completas el cuadro muy bien.
  • 50 puntos: Si es dudoso o alguien más hizo lo mismo.
  • 200 puntos: Si fuiste el único en completar el cuadro.
  • 0 puntos: Si no completaste el cuadro o incumpliste las reglas.
